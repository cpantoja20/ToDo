# ToDo
Basic To-do App Python Flask, React and MyQL


a) Flask

    1) Install tox
        "pip install tox"

    2) Run Tox
        "make test"
        
    3) Activate venv
        "source .tox/venv/bin/activate"

    4) Run project
        "make run"

    5) Acess to http://127.0.0.1:5000/


b)Database

    1) python
        from Api.app import db
        db.create_all()
        exit()

c) React

    1) Go to the folder 
        "cd Client/todo"

    2) Run project
        "npm start"

    3) Acess to http://localhost:3000/
